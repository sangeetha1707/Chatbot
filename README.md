# Gemini AI Chatbot

A modern chatbot application built with Vite+React and Bootstrap, powered by Google's Gemini 2.0 Flash model.

## Features

- 💬 Real-time chat interface
- 🤖 Powered by Gemini 2.0 Flash AI model
- 📱 Fully responsive design
- ⌨️ Markdown support for formatted responses
- 🎨 Modern UI with Bootstrap
- ⚡ Fast development with Vite
- 🔄 Typing indicators
- 🧹 Clear chat functionality

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Google Gemini API key
